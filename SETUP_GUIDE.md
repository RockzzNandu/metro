## 1. 📧 Linking Gmail for Real OTP Dispatch

To send actual emails from your localhost (XAMPP), it is highly recommended to use **PHPMailer**.

### Step A: Install PHPMailer
1. Download [PHPMailer](https://github.com/PHPMailer/PHPMailer) or use Composer:
   ```bash
   composer require phpmailer/phpmailer
   ```
2. Place the PHPMailer files in the `vendor/` directory.

### Step B: Configure SMTP in `includes/auth_functions.php`
Replace the `sendOTPEmail` function with the following logic:

```php
// Example logic for PHPMailer integration
use PHPMailer\PHPMailer\PHPMailer;
use PHPMailer\PHPMailer\Exception;

function sendOTPEmail($email, $otp) {
    $mail = new PHPMailer(true);
    try {
        $mail->isSMTP();
        $mail->Host       = 'smtp.gmail.com';
        $mail->SMTPAuth   = true;
        $mail->Username   = 'YOUR_GMAIL_ADDRESS@gmail.com'; // Your Gmail
        $mail->Password   = 'YOUR_APP_PASSWORD';           // Gmail App Password
        $mail->SMTPSecure = PHPMailer::ENCRYPTION_STARTTLS;
        $mail->Port       = 587;

        $mail->setFrom('noreply@metroexpress.com', 'Metro Express AI');
        $mail->addAddress($email);
        $mail->isHTML(true);
        $mail->Subject = "Your Access Code: $otp";
        $mail->Body    = "Your verification code is <b>$otp</b>. It expires in 10 mins.";

        $mail->send();
        return true;
    } catch (Exception $e) {
        return false;
    }
}
```
> **Note:** You must generate a "Google App Password" from your Google Account Security settings to bypass 2FA.
